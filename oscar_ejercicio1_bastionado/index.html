- Entrar en UEFI

Para poder entrar en UEFI, deberemos apagar el equipo. Una vez apagado lo arrancaremos y pulsaremos la tecla F2 (reiteradas veces para asegurar...) Y nos saldrá un menu principal con información característica de nuestro portatil. Aqui existen varias cosas para securizar de la mejor manera posible nuestro portátil:

  • Contraseña para entrar en UEFI (HIPERVISOR Y USUARIO)
  • Habilitar modo seguro
  • Contraseña en el BOOT

    Contraseña UEFI

    Por defecto, la UEFI no trae contraseña para protegerla por ende es muy fácil que cualquier usuario entre y pueda tocar alguna configuración que podría peligrar nuestros datos...Así que le añadiremos una contraseña. A continuación explicaré las diferencias entre Supervisor y User

    Contraseña Supervisor

    Esta contraseña protege todo lo relacionado con la UEFI, configuración avanzada, Arranque del sistema, seguridad, entrada al cualquier menu... Es la "contraseña Maestra" de nuestra UEFI. Al crear una contraseña de Supervisor, habilitará la creación de Contraseña de usuario

    Contraseña Usuario

    User password, es una contraseña para entrar en la bios, como usuario normal la cual ofrece datos del sistema sin poder cambiar configuraciones avanzadas pero sí cambiar el sistema de arranque. Menu de un usuario normal en la bios

    Secure Boot

    El arranque seguro, es un modo para UEFI que trae Windows desde Windows 8, y que impide la ejecución de cualquier software no firmado o certificado en el arranque del sistema. Solo existen dos puntos, Habilitarlo y configuraciones sobre los certificados permitidos (No sé utiliza normalmente...). Y ya estaría protegido nuestro portátil
