  • Contraseña en el BOOT - Contraseña UEFI Por defecto, la UEFI no trae contraseña para protegerla por ende es muy fácil que cualquier usuario entre y pueda tocar alguna configuración que podría peligrar nuestros datos...Así que le añadiremos una contraseña. A continuación explicaré las diferencias entre Supervisor y User - Contraseña Supervisor Esta contraseña protege todo lo relacionado con la UEFI, configuración avanzada, Arranque del sistema, seguridad, entrada al cualquier menu... Es la "contraseña Maestra" de nuestra UEFI. Al crear una contraseña de Supervisor, habilitará la creación de Contraseña de usuario - Contraseña Usuario User password, es una contraseña para entrar en la bios, como usuario normal la cual ofrece datos del sistema sin poder cambiar configuraciones avanzadas pero sí cambiar el sistema de arranque. Menu de un usuario normal en la bios - Secure Boot software no firmado o certificado en el arranque del sistema. Solo existen dos puntos, Habilitarlo y configuraciones sobre los certificados permitidos (No sé utiliza normalmente...). Y ya estaría protegido nuestro portátil
